CASOS DE GIT. Regresar

Subir Proyecto a Github y a gh-pages

crear y subir repositorio github
  • Primeros Creas el repositorio en la pagina web (github.com)
  • • Luego vas a gitbash
  • git init
  • git add .
  • git commit -m "COMENTARIOS"
  • • Con Estos Pasos queda en Local
    Subir Repositorio github
  • • git remote add origin https://github.com/apere7/TRANSPORTES.git
  • • git push -u origin master  
  • Con Estos Pasos queda en WEB
    Subir Repositorio al hosting gh-pages
  • Subir gh-pages
  • • Una vez el proyecto creado Vas a -->Settings arriba a mano derecha y das click
  • • bajas hasta -->github pages debe decir (master branch)
  • • Luego vamos al repositorio y en la pestaña (branch master) das click
  • • colocas gh-pages
  • • vuelves a setting vas a github pages y ahi esta la ruta para ver la pagina web  
  • CON ESTO YA tienes el proyecto cargado en gh-pages
    Subir Cambios al hosting gh-pages
  • • git branch (debera tener el *gh-pages y no el master)
  • • git checkout gh pages
  • • git push origin gh-pages
